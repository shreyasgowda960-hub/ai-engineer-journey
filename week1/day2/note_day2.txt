
IF / ELIF / ELSE
    if condition:
        ...
    elif other_condition:
        ...
    else:
        ...

    only one branch runs - Python checks if, then elif's in order, runs the
    first one that's True, else runs else if none matched.
    condition must eventually be True/False (or something Python can treat as
    truthy/falsy).

    age=18
    if age>=18:
        print("adult")
    elif age>=13:
        print("teen")
    else:
        print("child")


FOR LOOP
    for item in sequence:
        ...

    runs once per item in whatever you're looping over (list, string, range, etc.)

    for i in range(5):
        print(i)          -> 0 1 2 3 4


RANGE()
    range(stop)              -> 0 to stop-1
    range(start, stop)       -> start to stop-1
    range(start, stop, step) -> start to stop-1, skipping by step

    range(5)        -> 0,1,2,3,4
    range(2,5)      -> 2,3,4
    range(0,10,2)   -> 0,2,4,6,8
    range doesn't include the stop value itself.


WHILE LOOP
    while condition:
        ...

    keeps running as long as condition stays True. Must update something
    inside the loop that eventually makes it False, or it runs forever.

    n=0
    while n<5:
        print(n)
        n+=1              -> 0 1 2 3 4


BREAK / CONTINUE
    break    - exits the loop immediately, skips whatever's left in it
    continue - skips the rest of this iteration, goes to the next one

    for i in range(10):
        if i==5:
            break          -> stops entirely at 5, prints 0 1 2 3 4
        print(i)

    for i in range(10):
        if i%2==0:
            continue       -> skips even numbers, prints 1 3 5 7 9
        print(i)
